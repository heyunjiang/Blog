## 4 vue3 学习思考

vue3 学习可以在开发 vue3 相关项目时针对学习  
几年后，如果又出了 vue4，新人又是直接学习的 vue4，到时候也需要同步学习 vue4，那么 vue3 对于我而言，又有什么意义？我学习 vue3 又学习了什么？

1. 学习 vue3 基本原理，包括渲染流程、响应式原理、各 api 原理
2. 学习 ts 写法
3. 总结归纳前端框架流程
